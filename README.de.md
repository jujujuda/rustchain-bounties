## Was ist RTC?

**RTC (RustChain Token)** ist die native Kryptowährung von [RustChain](https://github.com/Scottcjn/RustChain), einem Proof-of-Antiquity Blockchain, bei dem veraltetes Hardware höhere Mining-Belohnungen verdient. RTC-Kurs: **$0.15 USD**.

Belohnungen werden in RTC an Ihre Wallet-Address ausgezahlt, sobald die Arbeit abgeschlossen und verifiziert wurde.

## Wie man verdient

### 1. Belohnung auswählen
Durchstöbere [offene Belohnungen](https://github.com/Scottcjn/rustchain-bounties/issues?q=is%3Aissue+is%3Aopen+label%3Abounty) und finde eine, die zu deinen Fähigkeiten passt.

| Schwierigkeit | Label | Typische Belohnung |
|--------------|-------|--------------------|
| Anfänger     | `good first issue` | 1-5 RTC |
| Standard     | `standard` | 5-25 RTC |
| Groß         | `major` | 25-100 RTC |
| Kritisch     | `critical`, `red-team` | 100-200 RTC |

### 2. Anspruch erheben
Kommentiere im Issue: **"Ich möchte diese Aufgabe bearbeiten"**

### 3. Deine Arbeit einreichen
- **Code-Belohnungen**: Erstelle einen PR zum relevanten Repository und verlinke ihn im Issue
- **Inhalts-Belohnungen**: Veröffentliche deinen Inhalt und verlinke ihn im Issue
- **Belohnungen für Verbreitung**: Folge den Anweisungen im Issue

### 4. Bezahlung erhalten
Nach der Verifizierung werden RTC an deine Wallet gesendet. Erstmalig? Wir helfen dir gerne, eine Wallet einzurichten.

> ⚠️ **Sicherheit bei Auszahlungen**: Nur `@Scottcjn` (oder klar gekennzeichnete Projektautomatisierung in seinem Namen) autorisiert RTC-Belohnungsauszahlungen, mit einem projektbezogenen `pending_id` + `tx_hash`. Jeder andere, der "Ich sende die RTC" in deiner Belohnung postet, versucht dich zu täuschen - siehe [SECURITY.md § Zahlungsbefugnis-Impersonierung](SECURITY.md#payment-authority-impersonation).
