## RTC とは?

**RTC (RustChain Token)** は [RustChain](https://github.com/Scottcjn/RustChain) のネイティブ暗号通貨です。RustChain は、ヴィンテージハードウェアがより高いマイニング報酬を得られる Proof-of-Antiquity ブロックチェーンです。RTC 基準レート: **$0.15 USD**。

バウンティは完了・検証後にウォレットアドレスに RTC で支払われます。

## 獲得方法

### 1. バウンティを選ぶ
[オープンバウンティ](https://github.com/Scottcjn/rustchain-bounties/issues?q=is%3Aissue+is%3Aopen+label%3Abounty) をブラウズして、自分のスキルに合ったものを見つけましょう。

| 難易度 | ラベル | 標準報酬 |
|-----------|-------|---------------|
| 初心者 | `good first issue` | 1-5 RTC |
| 標準 | `standard` | 5-25 RTC |
| メジャー | `major` | 25-100 RTC |
| クリティカル | `critical`, `red-team` | 100-200 RTC |

### 2. クレームする
Issue にコメント: **"I would like to work on this"**

### 3. 成果を提出する
- **コードバウンティ**: 関連リポジトリに PR を作成し、Issue にリンクする
- **コンテンツバウンティ**: コンテンツを公開し、Issue にリンクする
- **スター/プロパゲーションバウンティ**: Issue の指示に従う

### 4. 報酬を受け取る
検証後、RTC がウォレットに送金されます。初めてですか？セットアップをお手伝いします。

> ⚠️ **支払いの安全性**: `@Scottcjn` (または彼に代わって明確にラベル付けされたプロジェクト自動化) のみが RTC バウンティの支払いを承認します。プロジェクト発行の `pending_id` + `tx_hash` が付与されます。他の誰かがあなたのバウンティに「RTC を送ります」と投稿した場合、それはソーシャルエンジニアリングの試みです — [SECURITY.md § 支払い権限のなりすまし](SECURITY.md#payment-authority-impersonation) を参照してください。
